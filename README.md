# MICROCICLO APP




<p>Aplicación web para la gestión de microciclos de entrenamiento deportivo. Permite planificar y organizar los entrenamientos de una semana (microciclo), con soporte para múltiples sesiones diarias y exportación a PDF.</p>

<h3>Características</h3>
<ul>
  <li>Agregar y eliminar días de entrenamiento</li>
  <li>Programar hasta 4 sesiones por día</li>
  <li>Guardar y editar sesiones de entrenamiento</li>
  <li>Exportar microciclo completo a PDF</li>
</ul>


<h3>Tecnologías</h3>
<ul>
  <li>Bootstrap 5 - Estilos y componentes responsive</li>
  <li>jQuery - Manipulación del DOM y eventos</li>
  <li>JavaScript (ES6+) - Lógica de la aplicación</li>
  <li>jsPDF - Generación de documentos PDF</li>
</ul>

<h3>Cómo usar</h3>
<ol>
  <li>Abre <code>index.html</code> en un navegador web</li>
  <li>Haz clic en "Nuevo Registro" para agregar días al microciclo</li>
  <li>Completa las sesiones de entrenamiento para cada día</li>
  <li>Guarda cada día individualmente</li>
  <li>Exporta el microciclo completo a PDF cuando termines</li>
</ol>
